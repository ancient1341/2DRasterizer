Requirements to run:
-Gradle
-Java



Usage:
From Command line enter 2DRasterizer directory, then enter commands

gradle build

java -jar build/libs/2DRasterizer.jar



Controls:
WASD for movement
SPACEBAR to jump
Up/Down Arrows to look up and down
